# 🌸 Ciamis Jelita

**Ciamis Jelita** is an advanced AI-driven interactive application that combines modern web technologies with artificial intelligence to deliver a unique user experience. Built with **TypeScript** and **Vite**, it ensures lightning-fast performance and a robust development environment.

## 🚀 Features

- **AI Integration**: Leverages AI Studio capabilities for intelligent interactions.
- **Real-time Database**: Powered by Firebase Firestore for seamless data updates.
- **Responsive UI**: Built with React and optimized for all screen sizes.
- **Fast Development**: Powered by Vite for an optimized build process.

## 🛠️ Technology Stack

- **Framework**: React (with TypeScript)
- **Build Tool**: Vite
- **Backend**: Firebase (Firestore, Hosting)
- **State Management**: React Hooks

## 📦 Installation

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build
```

## 🛡️ Security & Rules

This project includes pre-configured Firebase security rules (`firestore.rules`) and indexes (`firestore.indexes.json`) to ensure data integrity and security.
